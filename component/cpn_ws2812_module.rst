Circular WS2812 LED Module
===========================================================

.. image:: img/c_ws2812_module.png
    :width: 40%
    :align: center

この円形 WS2812 LED モジュールには、12 個の個別アドレス指定可能な RGB LED が搭載されています。各 LED にはドライバーチップが内蔵されており、1 本のデータラインで 24 ビットのカラー制御を行うことができます。モジュールには VCC、GND、DIN、DOUT の 4 つのピンがあり、Raspberry Pi や Arduino などのコントローラーと簡単に接続できます。

**ピン概要**

* **V (VCC)**: 5V 電源入力。すべての LED に十分な電流を供給できる電源を使用してください。
* **G (GND)**: グラウンド。信号通信が正しく動作するよう、コントローラーの GND と接続する必要があります。
* **R (DIN)**: マイクロコントローラーからのデータを受信します。これは制御用の入力ピンです。
* **D (DOUT)**: 追加の LED リングや LED ストリップへデータを出力し、デイジーチェーン接続による拡張を可能にします。

**使用方法**

LED はデータを次の LED へ順番に転送します。これにより、各 LED を個別に制御することができ、カラーグラデーション、回転効果、さまざまなアニメーションパターンなどの演出を作成できます。  
このコンパクトな 12 LED リングは、ステータス表示、装飾用照明、ロボットプロジェクト、ウェアラブルデバイス、時計、小型表示装置やビジュアライゼーション用途に最適です。
