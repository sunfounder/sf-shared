Raspberry Pi 用電源（重要）
===============================

充電
-------------------

まずバッテリーケーブルを接続し、次に USB-C ケーブルを挿してバッテリーを充電します。  
充電器は別途ご用意ください。 **5V 3A の充電器** を推奨します。一般的なスマートフォン用充電器でも使用可能です。

.. image:: img/robot_hat_charge.png
    :width: 600
    :align: center

.. note::
    外部の Type-C 電源をロボットハットの Type-C ポートに接続すると、すぐに充電が開始され、赤色のインジケーターランプが点灯します。\
    バッテリーが満充電になると、赤色ランプは自動的に消灯します。


電源オン
----------------------

電源スイッチをオンにします。電源インジケーターランプとバッテリー残量インジケーターランプが点灯します。

.. image:: img/robot_hat_power.png
    :width: 600
    :align: center

数秒待つと、短いビープ音が鳴り、Raspberry Pi が正常に起動したことを示します。

.. note::
    バッテリー残量インジケーターランプが両方とも消灯している場合は、バッテリーを充電してください。  
    長時間のプログラミングやデバッグを行う場合は、USB-C ケーブルを接続してバッテリーを充電しながら Raspberry Pi を動作させることができます。
