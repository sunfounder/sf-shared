
Raspberry Pi 电源供电（重要）
=====================================================

充电
-------------------

插入电池连接线，然后插入 USB-C 线为电池充电。  
你需要自备充电器；建议使用 **5V 3A** 的充电器，常用的手机充电器也可以满足需求。

.. image:: img/robot_hat_charge.png
    :width: 600
    :align: center

.. note::
    将外部 Type-C 电源接入机器人 HAT 上的 Type-C 接口后，电池会立即开始充电，同时红色指示灯亮起。\
    当电池充满电后，红色指示灯会自动熄灭。


开机
----------------------

打开电源开关，电源指示灯和电池电量指示灯将会亮起。

.. image:: img/robot_hat_power.png
    :width: 600
    :align: center

等待几秒钟后，你会听到一声轻微的提示音，表示 Raspberry Pi 已成功启动。

.. note::
    如果两个电池电量指示灯都未亮，请先为电池充电。  
    当需要长时间进行编程或调试时，可以在给电池充电的同时插入 USB-C 线，以保持 Raspberry Pi 持续运行。
