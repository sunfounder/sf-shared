モバイル端末で Blynk を使う方法
===============================

.. note::

    データストリームは Blynk の Web 上でのみ作成できるため、まず Web で別のプロジェクトを参照してデータストリームを作成し、その後、以下のチュートリアルに従ってモバイル端末の Blynk でウィジェットを作成する必要があります。


#. モバイル端末で Google Play または App Store を開き、「Blynk IoT」（Blynk（レガシー）ではありません）を検索してダウンロードします。
#. アプリを開いたらログインします。このアカウントは、Web クライアントで使用しているアカウントと同一である必要があります。
#. 次に **Dashboard** に移動します（まだない場合は作成してください）。モバイル用と Web 用の **Dashboard** は互いに独立していることが確認できます。

    .. image:: img/APP_1.jpg

#. **Edit** アイコンをクリックします。
#. 空白エリアをクリックします。
#. Web ページと同じウィジェットを選択します。例えば **Joystick** ウィジェットを選択します。

    .. image:: img/APP_2.jpg

#. 空白エリアに **Joystick** ウィジェットが表示されるので、それをクリックします。
#. **Joystick** の設定画面が表示されます。Web ページで設定した **Xvalue** と **Yvalue** のデータストリームを選択します。なお、各ウィジェットは、各プロジェクト内の異なるデータストリームに対応している点に注意してください。
#. **Dashboard** ページに戻ると、必要に応じて **Joystick** を操作できるようになります。

    .. image:: img/APP_3.jpg
